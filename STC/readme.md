# Gear_Library/STC

主要存放在STC平台使用的轮子

| 文档版本名称简写 |                   解释                    |
| :--------------: | :---------------------------------------: |
|        A         | Alpha版本，功能不完善或未经测试或没有优化 |
|        R         |    Release版本，功能经过测试且正常运行    |

编译环境均为Keil® 4 C51环境
